FIDS (File-based image distribution system)

Intro

FIDS is a program for compressing binaries into image files, which can be shared as an image using an image sharing service like imgur and decoded.

Installation

Download the binary here.

If you would like to compile it yourself, open this project in Visual Studio and click "Compile".
